Four directions follow from the substrate, and skill induction leads. Repeated
command sequences are already visible in the reflex store, so skills can be
\emph{mined} from it: recurring \code{(request, command)} chains clustered into named,
editable procedures the user can invoke and the agent can propose, treating skill
synthesis as the plan-then-execute decomposition of long-horizon agents
\citep{erdogan2025planandact} and the terminal-native analogue of a skill library
(\S\ref{sec:related}). The rest follow in one breath: recognizing the shape of work
above the text of commands (a training run, its next step, its deviations)
\citep{wu2025osatlas}; semantic retrieval once mined skills widen the paraphrase gap
BM25 bridges, a broker-carried store (\S\ref{sec:broker}), and a redaction pass
(Ethics Statement) grown toward learned secret detection; and learned routing from the
log's per-task accept record, with watch summarizing an overnight run into memory
unprompted.
