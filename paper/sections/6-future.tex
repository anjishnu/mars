\section{Future Work}
\label{sec:future}

Four directions follow from the substrate: mining repeated command sequences into
editable procedures, treating skill synthesis as the plan-then-execute decomposition
of long-horizon agents \citep{erdogan2025planandact}; recognizing the shape of work
above the text of commands, a training run or a debug session, and flagging deviation
\citep{wu2025osatlas}; semantic retrieval and a broker-carried store
(\S\ref{sec:system}) once mined skills widen the paraphrase gap BM25 bridges, with
the redaction pass (Ethics Statement) growing toward learned secret detection; and
learned routing from the log's per-task accept/reject record, letting the watch
capability summarize an overnight run into memory unprompted.
